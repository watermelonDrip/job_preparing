# 剑指offer

从尾到头反过来打印出每个结点的值。

```python
# Definition for singly-linked list.
# class ListNode:
#     def __init__(self, x):
#         self.val = x
#         self.next = None

class Solution:
    def reversePrint(self, head: ListNode) -> List[int]:
        res = list()
        while head:
            res.append(head.val)
            head = head.next
        return res[::-1]
```
T: O(n)
S: O(n)
